# CardioPredict AI — Streamlit

Heart disease risk predictor using logistic regression trained on the Cleveland Heart Disease Dataset (UCI).

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Community Cloud (free)

1. Push this folder to a **public GitHub repo**
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Sign in with GitHub
4. Click **New app** → pick your repo → set `app.py` as the main file
5. Click **Deploy** — live in ~2 minutes ✅

Your app gets a free URL: `https://your-app-name.streamlit.app`
